Authentication

Option 1: JWT Token (recommended)

Exchange your API key for a short-lived JWT (3 hours). Best for client-side apps — never expose your raw API key in frontend code.

POST /v1/auth/token
{"api_key": "YOUR_API_KEY"}

→ {"token": "eyJ...", "expires_in_seconds": 10800}

Option 2: Raw API Key (server-to-server)

Pass the API key directly as Bearer token. Only use this from backend code.

Authorization: Bearer YOUR_API_KEY

Rate limit: 30 requests per minute per key/token.

Sampling

FieldTypeDefaultRangeDescription
temperaturefloat0.70–2Controls randomness. Lower = more focused, higher = more creative. 0 = deterministic.
top_pfloat0.90–1Nucleus sampling. Only consider tokens with cumulative probability ≤ top_p. 1 = disabled.

Penalties

FieldTypeDefaultRangeDescription
repeat_penaltyfloat1.00+Penalize repeated token sequences. 1.0 = disabled. Try 1.1–1.3.
presence_penaltyfloat0.00–2Penalize tokens already in the output. Encourages new topics.
frequency_penaltyfloat0.00–2Penalize tokens by frequency. Reduces repetition.

Python

OpenAI SDK

from openai import OpenAI

client = OpenAI(
    base_url="http://localhost:8000/v1",
    api_key="YOUR_API_KEY",
)

response = client.chat.completions.create(
    model="Falcon3-7B-Instruct-1.58bit",
    messages=[
        {"role": "system", "content": "You are a helpful assistant."},
        {"role": "user", "content": "What is Bitcoin?"},
    ],
    max_tokens=200,
    temperature=0.7,
    top_p=0.9,
)
print(response.choices[0].message.content)

requests + JWT

import requests

BASE = "http://localhost:8000"

# Step 1: Get token (cache this, refresh hourly)
token = requests.post(f"{BASE}/v1/auth/token",
    json={"api_key": "YOUR_API_KEY"}).json()["token"]

# Step 2: Chat
resp = requests.post(f"{BASE}/v1/chat/completions",
    headers={"Authorization": f"Bearer {token}"},
    json={
        "model": "Falcon3-7B-Instruct-1.58bit",
        "messages": [{"role": "user", "content": "Hello!"}],
        "temperature": 0.7,
        "top_p": 0.9,
    }).json()

print(resp["choices"][0]["message"]["content"])

JavaScript

OpenAI SDK

import OpenAI from 'openai';

const client = new OpenAI({
  baseURL: 'http://localhost:8000/v1',
  apiKey: 'YOUR_API_KEY',
});

const resp = await client.chat.completions.create({
  model: 'Falcon3-7B-Instruct-1.58bit',
  messages: [{ role: 'user', content: 'What is Bitcoin?' }],
  temperature: 0.7,
  top_p: 0.9,
});
console.log(resp.choices[0].message.content);

fetch + JWT

const BASE = 'http://localhost:8000';

// Step 1: Get token
const { token } = await fetch(`${BASE}/v1/auth/token`, {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({ api_key: 'YOUR_API_KEY' }),
}).then(r => r.json());

// Step 2: Chat
const data = await fetch(`${BASE}/v1/chat/completions`, {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'Authorization': `Bearer ${token}`,
  },
  body: JSON.stringify({
    model: 'Falcon3-7B-Instruct-1.58bit',
    messages: [{ role: 'user', content: 'Hello!' }],
    temperature: 0.7,
  }),
}).then(r => r.json());

console.log(data.choices[0].message.content);

cURL

# Get token
TOKEN=$(curl -s -X POST http://localhost:8000/v1/auth/token \
  -H "Content-Type: application/json" \
  -d '{"api_key":"YOUR_API_KEY"}' | python3 -c "import sys,json;print(json.load(sys.stdin)['token'])")

# Chat
curl -X POST http://localhost:8000/v1/chat/completions \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer $TOKEN" \
  -d '{
    "model": "Falcon3-7B-Instruct-1.58bit",
    "messages": [{"role": "user", "content": "Hello!"}],
    "temperature": 0.7,
    "top_p": 0.9,
    "max_tokens": 200
  }'
